React 기능만 골라

5일안에 마스터하기

 1) React의 주요 기능 중 업무에 바로 적용해볼 수 있는 주제를 선정해 현업 프론트엔드 엔지니어분들과 함께 학습합니다.

 

2) React 문법을 기본으로 알고 있는 분들을 대상으로 하며, React 기본 문법에 대한 내용은 설명하지 않습니다.  

 

3) React 주요 기능에 대한 높은 이해와 원활한 클래스 운영을 위해 5일 모두 참석하시길 권장드립니다. 

 

4) 반드시 아래 프로그램 설치 후 참석해 주시기 바랍니다.

 - yarn

→ 금토동 삼거리에서 서울,성남시청 방면 좌회전

→ 500m 직진 → 목적지(좌측) 도착

 


2. 제2경인고속도로 이용
동판교 IC에서 양재, 내곡 방면 진출

→ 시흥지하차도 옆길 → 우회전 후 다음 신호에서 유턴

→ 1km 직진 → 목적지(우측) 도착

 


3.서울외곽순환도로 이용

→ 3km 직진 → 목적지(좌측) 도착
